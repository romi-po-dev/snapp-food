# Snapp Food 🍕

A food delivery desktop application built with Python.

## About
Snapp Food is a fully functional food delivery app where customers 
can browse restaurants, place orders, and managers can manage 
their menu and track orders.

## Features
- User authentication (login & register)
- Customer and Manager roles
- Restaurant and menu management
- Order placement and tracking
- Menu item categories
- Data persistence with SQLite database

## How to Run
1. Make sure you have Python installed
2. Clone this repository
3. Install dependencies:
   pip install tkinter
4. Run the app:
   python SnappFood.py

## Built With
- Python
- Tkinter (GUI)
- SQLite (Database)
